# Installation

You can download the plugin from the [downloads page](/download) or from [GitHub](https://github.com/skriptlibrary/SkLibrary/releases).

Place the downloaded `.jar` file into you server's `plugins` folder and restart the server.

After the server has restarted, SkLibrary is ready to use!
